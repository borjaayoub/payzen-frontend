# Design System

This directory contains the design system components and styles for the application.

## Structure

- `styles/` - Global design system styles
- `components/` - Reusable UI components
  - `button/` - Button component
  - `input/` - Input component
  - `card/` - Card component
  - `alert/` - Alert component
  - `badge/` - Badge component
  - `typography/` - Typography component
  - `form/` - Form component

## Usage

Import components from the design system:

```typescript
import { DsButtonComponent } from './design-system/components/button/ds-button.component';
```
